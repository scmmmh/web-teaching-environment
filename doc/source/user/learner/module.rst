Module Content
==============

Each module you :doc:`take <take_module>` will consist of one or more sections. These sections
can cover the full range of content, including lectures, tutorials, exercises,... You can access
these sections by clicking on the module's title in the list of :doc:`your modules <my_modules>`.
Then select the section you want to view by clicking on it's title. Currently there are two different
types of sections:
