Exercises
=========

Each module consists of multiple :doc:`tutorials <tutorials>` and exercises.
When you select an exercise in a module, you are shown the exercise's details
page, which consists of a brief summary of the exercise and then the exercise's
list of tasks.

For each task, you will see the title and a brief summary of the task. Click on
the task's title to undertake that task. The Web Teaching Environment supports
the following types of tasks:
