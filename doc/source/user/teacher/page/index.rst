Page
====

Pages contain the actual content in the Web Teaching Environment.

The following actions are available on pages via the :icon:`fi-list` menu icon, both on
the page and also for each page on the list of your pages in a part:

+------------------------+----------------------+-------------------------+
| Icon                   | Label                | Action                  |
+========================+======================+=========================+
| :icon:`fi-pencil`      | Edit                 | :doc:`edit`             |
+------------------------+----------------------+-------------------------+
|                        | Add Page Before      | :doc:`new`              |
+------------------------+----------------------+-------------------------+
|                        | Add Page After       | :doc:`../asset/new`     |
+------------------------+----------------------+-------------------------+
|                        | Add Asset            | :doc:`../asset/new`     |
+------------------------+----------------------+-------------------------+
|                        | Export               | :doc:`../import_export` |
+------------------------+----------------------+-------------------------+
|                        | Download             | :doc:`../download`      |
+------------------------+----------------------+-------------------------+
| :icon:`fi-trash`       | Delete               | :doc:`delete`           |
+------------------------+----------------------+-------------------------+
