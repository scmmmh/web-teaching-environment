Changing the StatusÂ¶

A module or part’s status define which users can access that module or part’s content. In the WTE the following three status are available:

Available
The module or part is available and can be accessed by any user who is registered for the module itself or the module the part belongs to. A module that is available will also be listed on the list of available modules. A part that is available will be listed on the module’s page.
Unavailable
The module or part is not available. Only the module’s owner and any users that have the “tutor” role see and can access the module or part. For all other users an unavailable module will not be listed on the list of available modules and an unavailable part will not be listed on the module’s page.
Archived
The module has been archived and is not open for further registration. All users who have registered for the module continue to have full access, but no new users can register for it.

To change the status of a module or part, move your mouse over the icon for the module or part and then:

  • click on the Make available item to change the status to available
  • click on the Make unavailable item to change the status to unavailable
  • click on the Archive item to change the status to archived (only for modules)
