Creating a new Asset¶

To create a new asset, move your mouse over the icon and then select the Add Asset item.

You can then either select the file to upload as the new asset by clicking on the “Browse” button or by dragging a file into the browser window from your computer.

If you use the “Browse” button, then you can overwrite the filename that is used in the WTE for the asset by entering it into the “Filename” field. To create the new asset click on the Add button.

If you dragged a file into the browser window, then this will be uploaded immediately and the new asset created without any further interaction.

Creating assets in the content editor¶

You can also create assets and immediately insert the into the content of a module, part, or page. To do so open the content editor and then simply drag the file you wish to add as an asset into the editor window. This will automatically create a new asset and then insert the necessary :asset:`XX` to link the new asset.
