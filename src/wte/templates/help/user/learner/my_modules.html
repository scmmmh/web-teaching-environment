To access the list of modules that you are taking, move the mouse over the Your Name link in the top-right corner and then select the Modules option from the menu.

To access each module’s content, simply click on the title of the module. You can also download the module’s content by moving the mouse over the icon. Then select the Download item from the drop-down menu. This will generate a ZIP file with all the files you need to work on the module offline.
