wte.helpers – Helper functionality for views¶

The helpers module imports various helper modules that can be used in the templates via the h variable. For details of the functions available consulte the following documentation: pywebtools.text, wte.helpers.auth, and wte.helpers.form.

wte.helpers.version()[source]¶
